スマホでの開き方

1. AirDrop（iPhone/Mac のみ）
このHTMLファイルをAirDropでiPhoneに送り、Safariで開く。

2. GitHub Pages（おすすめ）
GitHubにファイルをアップすると固定URLでどの端末からもアクセス可能。

3. Dropbox / Google Drive
クラウドに保存し、スマホのブラウザで開く。

Firebase 設定（クラウド同期）

未設定
設定方法（5分でできます）
  1. console.firebase.google.com を開く
  2. 「プロジェクトを追加」で新規作成（無料）
  3. 左メニュー「Firestore Database」→「データベースの作成」→「テストモードで開始」
  4. 「プロジェクトの設定」→「ウェブアプリを追加」(</>)
  5. 表示された firebaseConfig の { } 全体をコピー
  6. 以下に貼り付けて「保存して接続」

同期キー

PC・スマホで同じキーを使うとデータが共有されます。
Firebase設定後にキーをコピーし、別の端末で「別のキーで引き継ぐ」から貼り付けてください。

自動同期URL（おすすめ）

下のボタンでURLをコピーし、PCのブラウザで開くだけで自動設定＆リアルタイム同期が始まります。

上の「Firebase設定」を先に行うと、ワンクリックで他の端末と自動同期できるURLを発行できます。

手動転送（Firebase不要）

①スマホで「転送コードをコピー」→ ②PCで貼り付けて「インポート」
